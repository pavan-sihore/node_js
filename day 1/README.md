## 1. Create a project folder

```bash
mkdir my-node-app
cd my-node-app
```

## 2. Initialize a Node.js project

```bash
npm init -y
```

## 3. Add express package

```bash
npm install express
```

## 4. Create a project structure

express-app/
├── public/
│   └── index.html
└── server.js

## 5. Run your server

```bash
node server.js
```
